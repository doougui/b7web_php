CTRL + P = Navega por arquivos
CTRL + G = Navega por linhas
CTRL + R = Navega por funções, classes css, etc
CTRL + M = Vai para o fechamento (CSS, if em PHP/JS)
CTRL + SHIFT + M = Seleciona tudo que está dentro do fechamento (CSS, if em PHP/JS)
CTRL + SHIFT + P + Snippets ou "lorem" + TAB = Lorem Ipsum automático
CTRL + F = Procura se tem o que eu digitei no arquivo
CTRL + SHIFT + F = Procura se tem o que eu digitei em todos os arquivos
ALT + F3 = Seleciona todas as palavras iguais a que está selecionada
CTRL + Z = Desfaz
CTRL + SHIFT + Z = Refaz
CTRL + D = Seleciona as palavras iguais a que está selecionada (uma a uma)
CTRL + H = Replace
CTRL + A = Seleciona tudo dentro do documento.
CTRL + SHIFT + A = Seleciona tudo dentro de uma TAG (only HTML)
CTRL + L = Seleciona uma linha inteira
CTRL + SHIFT + J = Seleciona por identação
CTRL + K + K = Exclui uma linha sem selecionar
CTRL + C = Copia uma linha inteira sem precisar selecionar
CTRL + SHIFT + V = Cola com a identação correta onde o cursor estiver
ALT + SHIFT + 2 = Trabalha com duas janelas na mesma aba
